GrantResourceOwnerCredentials(OAuthGrantResourceOwnerCredentialsContext)

This method validates the user asking for a token from the authorization endpoint, by checking the Username and Password fields of the context provided. When the user is validated the username is added to the token in order to determine which user is asking for a protected resource in the UserController.
