Interface IAuthenticatedUser

This is the interface for extracting information from the HttpContext. The AuthenticatedUser and a fake version implements this interface. This interface makes it possible to take advantage of polymorphism, making it easy for the controllers to invoke the actual AuthenticatedUser for regular operations or the fake on for unit testing.
