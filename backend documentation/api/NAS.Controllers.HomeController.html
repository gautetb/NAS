Index(String, String, String, Nullable<Int32>)

This methods shows the front page of the admin site. It includes a list of all users, with options to:

  • Sort users.
  • Search based on names.
  • Edit users.

Declaration

Type Name Description
System.String searching

System.String sort

String to determine in what order the list is supposed to be returned (and showed) in.
System.String currentFilter

Current filter from the viewbag to keep the search string when changing pages browsing through the users.

Which page the admin is currenty on.
Returns

System.Web.Mvc.ActionResult

Returns a paged list of the users registered in the database.
Remarks

The view is a list of all the users show 10 at the time which are shown in different pages determined by the page number.
