Class NasBLL

This is the business logic layer of the application. It contains all the evaluation of the information received from Visma as well as sending requests to the data access layer.

Remarks

This class handles all the data requests sent to both Vismas servers as well as the database. It also updates the user base from the membership base in Visma eAccounting

The class can also determine the payment status and type of membership of the members obtained from Visma eAccounting
